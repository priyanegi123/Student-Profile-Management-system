# Student-Profile-Management-system
The Student Profile Management System is a desktop application developed using Java Swing to manage student information efficiently. It allows users to add, update, search, and delete student records through an interactive and user-friendly interface. The system displays data in tabular form and provides easy navigation for managing multiple student profiles. It is designed with modular Java code, ensuring scalability and maintainability. This project demonstrates practical implementation of CRUD operations, event handling, and GUI design concepts in Java, making it ideal for academic and administrative purposes.
